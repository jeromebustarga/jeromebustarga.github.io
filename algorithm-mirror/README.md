# The Algorithm Mirror

Web application for visualizing YouTube algorithmic influence.

## How to Run

1. Open `index.html` in any modern browser (Chrome, Firefox, Safari)
2. Click "Choose File" and upload a YouTube Takeout JSON file
3. Or click "Try Sample Data" to use generated example data

## Features

- **Hybrid AI Categorization**: 91.2% accuracy with cost optimization
- **Particle System**: Physics-based visualization of content clustering
- **Pixel Portrait**: Chronological grid showing viewing patterns
- **Abstract Art**: 8 generative algorithms for aesthetic visualization
- **Privacy-First**: All processing happens client-side

## Files

- `index.html` - Main application page
- `main.js` - Application controller
- `dataProcessor.js` - Data processing and AI categorization
- `metrics.js` - Algorithmic influence metrics
- `visualizations.js` - Particle system and pixel portrait
- `aiInsights.js` - AI-powered analysis using Gemini API

## Requirements

- Modern web browser with JavaScript enabled
- No server or installation required
- Optional: Gemini API key for AI categorization (embedded by default)
