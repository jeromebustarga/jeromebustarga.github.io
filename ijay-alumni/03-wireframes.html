Stacked card layout for directory and centered profile view on mobile devices.

Responsive Behavior

Desktop (1024px+)

Two-panel layout with grid and preview side-by-side. 3-column alumni grid. Full sidebar on profile page.

Tablet (768px - 1023px)

Stacked panels with preview below grid. 2-column alumni grid. Sidebar collapses above main content.

Mobile (<768px)

Single column list cards. No preview panel - tap goes direct to profile. Centered profile layout with stacked sections.
